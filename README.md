## 项目名称

 一个文件管理系统

## 项目简介

 建立一个文件管理系统能实现下列功能
- 实现用户的相关功能，如登陆等内容
- 实现用户的增删改查
- 能够实现文件的上传与下载
- 后续逐步完善更新
## 项目说明
- java版本为11
- ant的时候会出现很多警告，不影响正常使用。
- 上传的文件放在WEB-INF下的files目录下，还有一个tempDirectory用于临时存储文件，ant的时候注意这两个目录不能为空。
